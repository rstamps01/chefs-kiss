# Chef's Kiss Branding Package

This comprehensive branding package contains all the assets you need to establish a consistent and professional brand identity for Chef's Kiss across digital and print media.

## Package Contents

This package includes two complete sets of assets: **black** (for light backgrounds) and **white** (for dark backgrounds).

### 1. Logos (`/logos/`)

- **Black Logos**: `primary_logo.png`, `horizontal_logo.png`, `stacked_logo.png`, `wordmark_only.png`
- **White Logos**: Located in `/logos/white/`, includes `primary_logo_white.png`, `horizontal_logo_white.png`, etc.

### 2. Icons (`/icons/`)

- **Black Icons**: `app_icon_1024.png`, `ios_icon_*.png`, `android_icon_*.png`
- **White Icons**: Located in `/icons/white/`, includes `app_icon_1024_white.png`, `ios_icon_*_white.png`, etc.

### 3. Web Assets (`/web/`)

- **Black Web Assets**: `website_header_logo.png`, `favicon_*.png`
- **White Web Assets**: Located in `/web/white/`, includes `website_header_logo_white.png`, `favicon_*_white.png`

### 4. Social Media (`/social_media/`)

- **Black Social Media Assets**: `facebook_profile.png`, `twitter_header.png`, `linkedin_banner.png`
- **White Social Media Assets**: Located in `/social_media/white/`, includes `facebook_profile_white.png`, etc.

### 5. Print (`/print/`)

- **Black Print Asset**: `silkscreen_vector.png`
- **White Print Asset**: Located in `/print/white/`, `silkscreen_vector_white.png`

### 6. Brand Guide (`/brand_guide/`)

- **style_guide.md** - Complete brand style guide with usage guidelines for both black and white versions.

## Quick Start Guide

- **For Light Backgrounds**: Use the assets in the main directories (`/logos`, `/icons`, etc.).
- **For Dark Backgrounds**: Use the assets in the `white/` subdirectories.

## File Formats

All assets are provided in **PNG format with transparent backgrounds**.

## Color Specifications

- **Black**: #000000
- **White**: #FFFFFF

## Usage Guidelines

Please refer to the complete **Brand Style Guide** (`/brand_guide/style_guide.md`) for detailed usage guidelines.
